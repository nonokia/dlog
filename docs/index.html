dlog

An agent-first decision log that sits alongside Git. Git records what changed; dlog records the why — rationale, rejected alternatives, assumptions/invariants, and the original instruction — so AI agents can reconstruct context across sessions and multi-agent hand-offs.

Agents can discover dlog via ARD (/.well-known/ai-catalog.json). The catalog points at the agent skill doc, which covers install and usage.

Source and docs: github.com/nonokia/dlog.
